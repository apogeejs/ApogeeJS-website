Apogee - Interactive Programming

Apogee is an easy-to-use javascript programming environment that allows you to do interactive programming. It is based on the idea of a spreadsheet but it has all the power of standard programming. Watch the video below for an introduction.

For more help on using Apogee, go to Getting Started.

What can I do with Apogee?
  • Data Wrangling/Data Munging - Manipulating data, converting data format, filtering data, editing data programmatically. It is great for writing quick and dirty code you only plan on using once. It is especially good for working with JSON data, including converting to or from Spreadsheet table format or plain text.
  • Analysis/Reporting - Apogee is similar to a Jupyter notebook in the sense it uses literate programming. As with a notebook it can be used for analysis and writing reports. Apogee is not however geared towards the data scientist. It is more suited to information presentation and business applications.
  • Web Client Applictions - Apogee can be used to write application code for single page web applications. Follow the link to learn more about Web Client Applications.
  • Web Services - Apogee can also be used to code web services. Follow the link to learn more about Web Services.

Implementation

The Apogee web application runs completely in the browser and is written in javascript. The user scripting language is also javascript. Alternative implementations in other languages like Python would also work well.

A Electron desktop version of Apogee is also available for download. See the download page.

Because Apogee is uses javascript, the data format for data structures is JSON. This is a convenient because JSON is one of the most prevelent data formats, such as for metadata files and data transfer in web services.

Apogee if open source and hosted at https://github.com/sutter-dave/ApogeeJS.

Compatibility

  • Desktop Chrome, Firefox and Edge browsers tested. Safari also supported. It requires ES6.
  • Internet Explorer not supported
  • Desktop/mouse support only. MObile/touch not supported.
