Apogee Application

Apogee Web Application Link (v0.9.4) - This is the Apogee application running in a web browser. No installation required.

Apogee Electron Application Download (v0.9.4) - This is the version of the Apogee application that is the same as the web version except it runs in Electron and allows you to save and open from your local file system. This should be used for any purpose aside from writing web services.
Install Instructions:

Go to Getting Started for help on using the Apogee application.

Apogee Web Services

Apogee Web Server Download (v0.9.4) - This is a Node.js server that creates web services from Apogee workspaces.

Apogee "Node.js" Electron Application Download (v0.9.4) - This is the version of the Apogee application that should be used for developing workspaces for the Apogee server.

For documentation on writing web services and for installation instructions: click here
